Parcel配置

2020-01-11 tools Parcel 评论

零配置

指令配置路径 ：https://www.parceljs.cn/cli.html

安装

npm install -g parcel-bundler

使用

对应html文件中引用入口js文件

//Vue 挂载
import App from './App';
new Vue({
    el: '#app',
    router,
    render: (h) => h(App)
});

babel配置

新建.babelrc文件

//es6配置文件，webpack会自动读取
{
    "presets": [
        ["env", {
          "modules": false,
          "targets": {
            "browsers": ["ie >=9"]
          },
          "useBuiltIns": true,
        }],
        ["es2015", {"loose": true}]
      ]
}

Sass配置

新建.sassrc.js文件，配置Sass全局变量

const path = require('path');
const cwd = process.cwd();
module.exports = {
    data: '@import "src/css/vars";', //全局文件路径
    includePaths: [
        path.resolve(cwd, 'node_modules'),
        path.resolve(cwd, 'src', 'scss')
    ]
};

postcss配置

新建文件：postcss.config.js

安装依赖 postcss-pxtorem

const pxtorem = require('postcss-pxtorem');
module.exports = {
  plugins: [
    require('autoprefixer')({
            "overrideBrowserslist": [
                "defaults",
                "not ie < 9",
                "last 20 versions",
                "> 1%",
                "iOS 7",
                "last 3 iOS versions"
            ]
    }),
    pxtorem({
        rootValue: 100,
        unitPrecision: 5,
        propList: ['*', '!border*', '!background*'],
        selectorBlackList: [".no-tr-"], //不对这些样式进行转换
        replace: true,
        mediaQuery: false,
        minPixelValue: 6
    })

  ]
}

运行

parcel index.html
